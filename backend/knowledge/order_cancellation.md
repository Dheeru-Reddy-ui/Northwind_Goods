# Order Cancellation

## Cancelling before shipment
You can cancel an order for a full refund **any time before it ships** — while
it's still in "processing." Just ask our support agent with your order number and
we'll cancel it and release any payment hold.

## After it ships
Once an order has shipped it can no longer be cancelled. Instead, you can:
- Refuse delivery, or
- Accept it and start a return within the 30-day window.
Our agent can help you set either of these up.

## Refund after cancellation
If you were already charged, the hold or charge is reversed within a few business
days after cancellation.
